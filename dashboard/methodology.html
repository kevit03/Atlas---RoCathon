Method

The shortest path from vector retrieval to final rank.

Scoring stack

Core formulas

01 Semantic retrieval

Similarity score

semantic_score =
  max(0, 1 - cosine_distance(query_embedding, creator_embedding))

Vector search returns the nearest creators by semantic similarity.

Longer explanation

Atlas embeds the query and creator documents, then uses cosine-distance nearest neighbor search in pgvector. The score is converted into a bounded similarity value for reranking.

02 Projected score

Business normalization

projected_normalized =
  (projected_score - 60) / 40

RoC already provides a bounded commercial signal from 60 to 100.

Longer explanation

The projected score is min-max normalized into the same 0 to 1 space as the semantic score so the hybrid blend remains interpretable.

03 Hybrid reranking

Official ranker

projected_normalized = (projected_score - 60) / 40

final_score =
100 * (
  0.45 * semantic_score +
  0.55 * projected_normalized
)

This is the official challenge formula and the score that should be treated as primary.

Longer explanation

The ranker uses a convex combination because projected score already carries meaningful business signal. Once both components are normalized, linear fusion preserves that magnitude instead of collapsing the decision into rank positions.

Appendix

Atlas screen

commercial_index = 0.65 * projected_normalized
                 + 0.20 * engagement_norm
                 + 0.15 * gmv_norm

relevance_score =
  (industry_match + query_overlap + audience_fit) / 3

atlas_score =
100 * (
  0.60 * relevance_score +
  0.40 * commercial_index
)

Atlas is the live demo score: three relevance signals are treated equally, then blended against a commercial quality block.

Longer explanation

Atlas is meant for exploration across the whole creator universe, not just the challenge output. Instead of assigning different hand-picked weights to every relevance feature, the model averages industry fit, query overlap, and audience fit equally. That relevance block is then blended 60/40 against commercial quality so fit still leads the screen without introducing arbitrary fine-grained coefficients.
