Atlas Method

Method

A short explanation of the score and the research behind it.

Scoring stack

Core formulas

Challenge engine

Official ranker

projected_normalized = (projected_score - 60) / 40

final_score =
100 * (
  0.45 * semantic_score +
  0.55 * projected_normalized
)

This is the official challenge formula: normalized semantics plus normalized projected value, using the recommended 0.45 / 0.55 starting blend.

Longer explanation

The ranker uses a convex combination because projected score already carries meaningful business signal. Once both components are normalized, linear fusion preserves that magnitude instead of collapsing the decision into rank positions.

Dashboard screening

Atlas screen

commercial_index = 0.65 * projected_normalized
                 + 0.20 * engagement_norm
                 + 0.15 * gmv_norm

relevance_score =
  (industry_match + query_overlap + audience_fit) / 3

atlas_score =
100 * (
  0.60 * relevance_score +
  0.40 * commercial_index
)

Atlas is the live demo score: three relevance signals are treated equally, then blended against a commercial quality block.

Longer explanation

Atlas is meant for exploration across the whole creator universe, not just the challenge output. Instead of assigning different hand-picked weights to every relevance feature, the model averages industry fit, query overlap, and audience fit equally. That relevance block is then blended 60/40 against commercial quality so fit still leads the screen without introducing arbitrary fine-grained coefficients.

Quiz advisor

Advisor overlay

advisor_score =
  atlas_score
  + objective_bias
  + persona_bias
  + posture_bias

objective_bias favors:
- GMV for sales
- semantic fit for loyalty
- reach and views for awareness

The advisor adds a light campaign bias on top of the base score.

Longer explanation

The quiz is not a second ranking system. It lightly tilts the shortlist based on campaign objective, voice, and rollout posture so the output feels closer to a planning recommendation than a static leaderboard.
